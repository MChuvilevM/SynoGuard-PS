## ᛟ Описание / Overview

**SynoGuard-PS** — инструмент мониторинга Synology NAS корпоративного уровня. Разработка велась исключительно в среде **PowerShell ISE**, что гарантирует стабильность в стандартных Windows-инфраструктурах. В версии **1.1.1** реализован бесшовный темный интерфейс через прямое взаимодействие с системными библиотеками (DWM API).

## ᚱ Технические особенности / Technical Highlights

*   ⚡ **Async Engine**: Параллельный сбор метрик через **PowerShell Runspaces** (неблокирующий UI).
*   🌑 **Total Black UI**: Кастомный WinForms с использованием **DWM API (WinAPI)** для глубокого черного режима.
*   🛡️ **Data Security**: Шифрование критических данных по алгоритму **AES**.
*   🩺 **Health Tracking**: Мониторинг **RAID 6**, задержек и S.M.A.R.T. параметров.
*   📦 **Enterprise Build**: Автономный x64 бинарный файл для мгновенного развертывания.
