## ᛟ Описание / Overview

**SynoGuard-PS** — это плод глубокой интеграции PowerShell 5.1 с системными API. Разработка велась в классической среде **ISE**, что гарантирует стабильную работу в стандартных Windows-инфраструктурах без необходимости установки дополнительных зависимостей. Приложение обеспечивает мгновенный мониторинг RAID-массивов и дисков без блокировки интерфейса.

## ᚱ Технические особенности / Technical Highlights

*   ⚡ **Async Monitoring Engine**: Использование **PowerShell Runspaces** для параллельного сбора данных.
*   🌑 **Immersive Dark UI**: Кастомный WinForms интерфейс с интеграцией **DWM API (WinAPI)** для глубокого темного режима.
*   🩺 **Health Tracking**: Мониторинг RAID, S.M.A.R.T. статусов и задержек в реальном времени.
*   🛡️ **Data Security**: Безопасная передача данных через REST API с использованием шифрования.
*   📦 **Enterprise Build**: Скомпилированный автономный x64 бинарный файл для быстрого развертывания.
