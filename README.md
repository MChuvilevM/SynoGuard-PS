## 📋 Описание / Overview

**[RU]** **SynoGuard-PS** — это профессиональный инструмент мониторинга Synology NAS, ориентированный на высокую производительность. Благодаря асинхронному движку на базе Runspaces, приложение обеспечивает мгновенный мониторинг RAID-массивов и дисков без блокировки интерфейса.

**[EN]** **SynoGuard-PS** is a high-performance Synology NAS monitoring tool. Powered by an asynchronous Runspace engine, it provides real-time RAID and disk health tracking with a non-blocking, responsive UI.

> [!TIP]
> **[RU]** Оптимизировано для сред с высокой доступностью (High-Availability).
> **[EN]** Optimized for High-Availability (HA) environments.

## ✨ Ключевые возможности / Key Features

*   **⚡ Async Monitoring Engine:** 
    *   (RU) Использование PowerShell Runspaces для параллельного сбора данных.
    *   (EN) High-performance background data retrieval using PowerShell Runspaces.
*   **🌑 Immersive Dark UI:** 
    *   (RU) Кастомный WinForms интерфейс с интеграцией DWM API (WinAPI) для глубокого темного режима.
    *   (EN) Custom WinAPI-integrated "Total Black" dashboard for maximum focus.
*   **🩺 Health Tracking:** 
    *   (RU) Мониторинг RAID, S.M.A.R.T. статусов и задержек в реальном времени.
    *   (EN) Real-time tracking of RAID status, disk S.M.A.R.T., and latency.
*   **📦 Enterprise Build:** 
    *   (RU) Скомпилированный x64 бинарный файл с профессиональной иконкой.
    *   (EN) Compiled standalone x64 binary for easy infrastructure deployment.
